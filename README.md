# CORRECTOR_DC_BY_SONET
Script en Python que corregeix les tasques del Depatarment Comercial. Cómo usarlo

# Menú interactivo
python3 corrector_empresaula.py

# O por línea de comandos
python3 corrector_empresaula.py todo /ruta/csvs 12/12/2025
python3 corrector_empresaula.py compras /ruta/csvs
python3 corrector_empresaula.py ventas /ruta/csvs 14/11/2025

Solo necesita pandas instalado (pip install pandas). Los CSVs deben estar en el mismo directorio con los nombres exactos del PDF.
